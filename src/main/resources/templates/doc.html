User - Support Get/Post

getData/getUser

此接口用于获取用户的登录信息
action = getUser
username = yuan
password = 123456

接口参数
username 用户名
password 用户密码

getData/addUser

此接口用于注册用户
action = addUser
username = demo
password = 123456

接口参数
username 用户名
password 用户密码

getData/updateUser

更新用户的注册信息
action = updateUser
userid = 12
username = yuan
password = 123456
age = 18
sex = 男
phone = 1888888888
email = 110112@163.com
hoby = 无
note = 无

接口参数
userid 用户的id
username 用户名
password 密码
age 年龄
sex 性别
phone 电话
email 邮箱
hoby 兴趣爱好
note 备注

getData/deleteUser


删除注册的用户 注意：此接口不对外公布
action = deleteUser
username = demo
password = 123456

接口参数
username 用户名
password 密码

Device - Support Get/Post

getData/getAllDevice


获取当前用户所有设备信息
action = getAllDevice
userid = 12
username = yuan

接口参数
userid 用户id
username 用户名

getData/addDevice


添加设备获取注册设备 必须要注册设备之后才能使得设备获取到信息
action = addDevice
userid = 12
username = yuan
devicename = demo
deviceaddre = 00:00:00:00
添加设备的时间是由app/其它端上传上来 传入是Long型的参数 addtime = 152355660000

接口参数
userid 用户id
username 用户名
devicename 设备名 (从设备上得到)
deviceaddre 设备地址 (从设备上得到)
addtime app传入给服务器Long型时间戳参数

getData/updatDevice


更新设备的注册信息
action = updateDevice
deviceid = 10002
userid = 1001
username = demo
devicename = demo
deviceaddre = 00:00:00:00
addtime = 152355660000

接口参数
deviceid 设备id
userid 用户id
username 用户名
devicename 设备名 (从设备上得到)
deviceaddre 设备地址 (从设备上得到)
addtime app传入给服务器Long型时间戳参数

getData/deleteDevice


删除注册设备
action = deleteDevice
deviceid = 10002
userid = 1001
username = demo

接口参数
deviceid 设备id
userid 用户id
username 用户名

Probe - Support Get/Post

getData/getProbe

获取设备探头数据
action = getProbe
deviceid = 1
probename=one
begintime = 1516528220580
endtime = 1516756515102

deviceid 设备的id
probename 探头的名称
